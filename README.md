# Medical & Nursing Terminology Database

## Project Overview
This repository contains a comprehensive collection of **medical and nursing words** generated from the **200,000+ existing medical terminology database** using the **modular block system** (Prefix + Root + Combining Vowel + Suffix).

## Structure
The data is organized by word length:
- `data/word-length-9/` - 9-character words (100 terms)
- `data/word-length-10/` - 10-character words (100 terms)
- `data/word-length-11/` - 11-character words (100 terms)
- `data/word-length-12/` - 12-character words (100 terms)
- `data/word-length-13/` - 13-character words (100 terms)
- `data/word-length-14/` - 14-character words (100 terms)
- `data/word-length-15/` - 15-character words (100 terms)
- `data/word-length-16/` - 16-character words (100 terms)
- `data/word-length-17/` - 17+ character words (100 terms)

## JSON Structure

Each entry contains:
```json
{
  "word": "Medical term",
  "length": 9,
  "prefix": "prefix or empty",
  "prefix_meaning": "Meaning of prefix",
  "root_1": "root component",
  "root_1_meaning": "Greek/Latin meaning",
  "combining_vowel": "vowel between roots",
  "root_2": "second root or empty",
  "root_2_meaning": "Meaning if applicable",
  "suffix": "suffix component",
  "suffix_meaning": "Meaning of suffix",
  "literal_translation": "Direct translation",
  "category": "Inflammatory condition / Cardiovascular / etc.",
  "brief_nursing_elaboration": "Clinical nursing perspective and interventions"
}
```

## Data Sources
- SNOMED CT (Systematized Nomenclature of Medicine)
- ICD-11 Medical Classifications
- NANDA International Nursing Diagnoses
- Medical terminology references

## Clinical Application
This database serves as an educational resource for:
- Nursing students and educators
- Clinical practitioners
- Healthcare professionals
- Medical terminology learners
